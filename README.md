# Scintigrami - kartice za vežbanje

Flash kartice za pripremu praktičnog dela ispita iz nuklearne medicine.
32 scintigrama izvučena iz materijala "Primeri opisivanja scintigrama na praktičnom delu ispita".

## Kako se koristi

1. Pogledaj scintigram (sliku).
2. Opiši ga u sebi po 4 tačke: naziv, radiofarmak, mehanizam nakupljanja, nalaz.
3. Tapni na sliku (ili dugme "Prikaži odgovor") da proveriš.
4. Idi na sledeću karticu strelicama ili dugmadima.

Dugme "Promešaj" nasumično meša redosled kartica za pravo vežbanje.
